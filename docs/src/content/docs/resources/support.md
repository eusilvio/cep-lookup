# Support

Need help with CEP Lookup? Here are your options.

## GitHub Issues

Report bugs and request features on GitHub:

[github.com/eusilvio/cep-lookup/issues](https://github.com/eusilvio/cep-lookup/issues)

### Reporting Bugs

Include:
- Your Node.js/Browser version
- CEP Lookup version
- Minimal reproduction code
- Expected vs actual behavior
- Error message and stack trace

### Requesting Features

Describe:
- Use case and motivation
- Proposed API
- Examples

## Documentation

Check the documentation first:

- [Quick Start](/getting-started/quick-start)

## Response Times

- **Critical bugs**: 24 hours
- **Regular issues**: 48-72 hours
- **Documentation**: 1-2 weeks
